# Security Policy

## Supported versions

This project is under active development. Security fixes are applied to the latest `main` branch first.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Use one of the following:

- GitHub Security Advisories for this repository
- Private contact to maintainers with reproduction details

When reporting, include:

- Affected platform and version
- Reproduction steps
- Potential impact
- Suggested mitigation if available

We will acknowledge reports as soon as possible and coordinate disclosure after a fix is available.
